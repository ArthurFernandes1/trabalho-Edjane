# trabalho-Edjane
Gerenciador de Produtos - CRUD com Autenticação
Aplicação web full-stack (frontend) desenvolvida em React + Vite + JavaScript para cadastro, visualização, edição e exclusão de produtos, com sistema de autenticação simples de usuários.
🎯 Objetivo do Projeto
Desenvolver um sistema completo de gerenciamento de estoque de produtos com interface moderna, responsiva e intuitiva, atendendo aos requisitos de um trabalho acadêmico de Desenvolvimento Web.
✨ Principais Funcionalidades
Autenticação de Usuário:

Cadastro e Login de usuários
Validação de e-mail duplicado no cadastro
Mensagens de erro claras (e-mail não encontrado / senha incorreta)
Proteção de rotas (acesso apenas para usuários logados)

Gerenciamento de Produtos (CRUD completo):

Cadastro de novos produtos com imagem (via URL)
Edição e exclusão de produtos
Listagem dinâmica em cards responsivos
Filtro por categoria
Ordenação por nome, preço e quantidade
Busca por nome do produto

Validações e Experiência do Usuário:

Todos os campos obrigatórios validados
Preço e quantidade com validação numérica
Substituição de alert() por modais bonitos customizados
Mensagem amigável quando não há produtos cadastrados ("Nenhum produto cadastrado ainda")
Confirmação de ações importantes via modal

🛠️ Tecnologias Utilizadas

React + Vite (JavaScript)
React Router DOM (navegação)
Tailwind CSS (estilização moderna e responsiva)
Lucide React (ícones)
localStorage (persistência de dados no navegador)

📱 Design e Usabilidade

Interface limpa, moderna e totalmente responsiva (mobile-first)
Layout profissional com cards e navegação intuitiva
Modais elegantes para feedback ao usuário
Experiência fluida tanto em desktop quanto em dispositivos móveis

🔧 Destaques do Projeto

Código organizado e componentizado
Boas práticas de React (hooks, roteamento protegido, estado local)
Sistema de autenticação funcional com persistência
Tratamento cuidadoso de erros e feedback visual ao usuário
