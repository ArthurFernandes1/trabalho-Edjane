# Documentação do Projeto

Este diretório reúne a documentação técnica e funcional do app **Gestão de Produtos**.

## Conteúdo

- `01-visao-geral.md`: objetivo do projeto, escopo e funcionalidades principais.
- `02-arquitetura-e-codigo.md`: organização de pastas, componentes e fluxo de dados.
- `03-fluxos-funcionais.md`: fluxo de autenticação e gerenciamento de produtos.
- `04-ferramentas-e-tecnologias.md`: stack utilizada e papel de cada ferramenta.

## Resumo rápido

- Frontend em `React + Vite` com JavaScript.
- Navegação com `react-router-dom`.
- Estilização com `Tailwind CSS`.
- Ícones com `lucide-react`.
- Persistência local com `localStorage` (usuários, sessão e produtos).
- Interface responsiva com modais customizados, listagem com filtros e CRUD completo.
